### 修改日志
* 添加命令：`extract_text_folder`和`insert_text_folder`，用于批量提取和导入指定ws2的text.txt。
* 添加命令：`decompile_folder`和`compile_folder`，用于批量反编译和编译ws2的opcodes.json。
* 添加菜单：`Encoding`下`Write JIS`和`Write GBK`，可以按GBK写入文本。
